# Dunamiss Email Signature

This project contains the email signature template for Dunamiss with a clean, minimalist design featuring a large logo.

## File Structure

```
dunamiss email signature/
├── index.html          # Main HTML file for the email signature
├── css/
│   └── style.css       # Stylesheet for the email signature
├── images/             # Folder for all image assets
│   ├── logo.png        # Company logo
│   ├── line.png        # Divider line
│   ├── fb.png          # Facebook icon
│   ├── ln.png          # LinkedIn icon
│   ├── tt.png          # Twitter icon
│   ├── yt.png          # YouTube icon
│   ├── it.png          # Instagram icon
│   ├── wa.png          # WhatsApp icon
│   └── pt.png          # Pinterest icon
└── js/                 # JavaScript files (if needed)
    └── script.js
```

## Features

- Clean, professional design
- Responsive layout
- Social media integration
- Easy to customize contact information

## Usage

1. Place your images in the `images/` folder with the appropriate names
2. Update the placeholder text in `index.html` with your actual information
3. Customize styles in `css/style.css` as needed

## Notes

- This signature is designed to work across major email clients
- All styling is contained in external CSS for easy maintenance
- Images should be optimized for email (small file sizes)
- The signature has been customized with DUNAMISS COSMETICS information
